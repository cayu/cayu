---
title: "Listar último mes de login de usuarios"
date: 2014-01-20T16:03:32-03:00
---

1.  Launch the Domino administrator client and click the Configuration tab.
2.  Click the Server option, followed by the Configurations option.
3.  Select the server in which you want to identify inactive users, and select the Edit Configuration option.
4.  Click the Basics tab and click the Enabled option to the right of the License Tracking heading.
5.  Click “Save and Close” to save the changes.
6.  Click the Files tab in the Domino administrator and select the userlicenses.nsf file to open it.
7.  View the last access date for each user. Users without a recent last access date are inactive.
