## VLAN

Con el concepto de VLAN (Red de área local virtual) se pueden crear redes lógicas independientes dentro de una misma red física. Nos permiten utilizar una sola interfaz de red para llevar tráfico de varias subredes.

Las VLAN’s las podemos basar en :

- Puerto físico
- Dirección MAC
- Dirección IP
- Protocolo
