#### Configuración Minicom

```ini>
=== Configuración Inicial ===


En el wizard inicial se deben definir estas contraseñas :

  * **Secret password**
  * **Enable password**
  * **Virtual terminal password**

Luego debemos definir el ingreso via red
<code>
#sh privilege
#enable
#config t
#interface vlan 1
#ip address 10.1.1.22 255.255.255.0
#ip default-gateway 10.1.1.1

#copy running-config startup-config
#write memory
```

Servicio SSH

```
config t
ip domain-name CAYU.COM.AR
crypto key generate rsa
-- 2048
ip ssh version 2

service password-encryption

(config)#line vty 0 4
(config-line)#transport input ssh
(config-line)#login local
```
