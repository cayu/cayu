## Motorola - Moto X

1.  Con el celular apagado
2.  Presionar y mantener la tecla de Bajar Volumen y el Botón Power por 3 seconds y soltar
3.  En el menu : “Boot Mode Selection Menu” seleccionar Recovery, subiendo y bajando el volumen para seleccionar y power para confirmar esa opción
4.  En la pantalla “Android system recovery” screen, presionar y soltar “Subir Volumen y el Boton Power”
5.  Seleccionar la opción “wipe data/factory reset”
6.  Seleccionar “Yes – delete all user data”


## Nokia - C3

### Soft Reset Nokia c3.

Esto lo que hace es restaurar archivos .ini desde la ROM.

¿Como lo hacemos?…. Ingresa en tu nokia c3 este código \*#7780#

### Hard Reset Nokia c3

Este es el más completo y BORRA TODO lo que tiene tu celular, incluye tambien el borrado de la tarjeta de memoria si esta insertada.

¿Como lo hacemos?…. Ingresa en tu nokia c3 este código \*#7370#

### Pasos para restaurar el nokia C3

**Menu \> Configuración \> Rest Conf Fabrica.**

Aparece un menú que dice “SOLO CONFIGURACIONES” o “TODO”

Cuando le demos click en cualquiera de las dos opciones nos va a pedir un código de seguridad. Este es el código 12345.

Aparece un cartel que dice restaurando configuraciones \> Se reinicia el celular c3.
