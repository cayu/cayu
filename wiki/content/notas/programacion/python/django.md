Django es un framework de desarrollo web, escrito en Python,siguiendo el Modelo–vista–controlador.

## Iniciar un proyecto

```
django-admin startproject proyecto
```

Start Server

```
python manage.py runserver 192.168.1.43:8000
```

Creacion de admin:admin

```
python manage.py createsuperuser
```
