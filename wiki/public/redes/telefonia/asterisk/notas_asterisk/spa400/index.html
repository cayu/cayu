    Configuracion#

    Opciones del Menú de la Utilidad de Configuración del SPA400

    Los ítems del menú avanzado para el SPA400 son configurados por medio de la Utilidad de Configuración basada en la red.

    Configuración: Configuración Básica
    Dato – Configuración de Red
    Dirección IP Dinámica Revise este botón para obtener los parámetros de red desde un servidor DHCP
    Dirección IP Fija Revise este botón para permitir que el SPA400 sea configurado con una dirección IP estática
    Máscara de Subred IP Máscara de subred
    Dirección IP de Puente de Acceso Dirección IP del Puente de Acceso/Router
    Dato – Dirección de Servidor de Nombre de Domino
    DNS Primario Dirección IP del servidor primario de nombre de dominio
    DNS Secundario Dirección IP del servidor secundario de nombre de dominio
    Dato – NTP
    Servidor NTP Dirección IP o FQDN de un servidor NTP
    Zona Horaria Seleccione la zona horaria

    Configuración: Interfaz SPA9000
    Dato – Identificación de Usuario SPA9000
    Identificación de Usuario La identificación de usuario del SPA9000. Debe ser idéntica a la identificación de usuario configurada en la solapa Línea # del SPA9000
    Descubrir Automáticamente Seleccione este botón si quiere que el SPA400 aprenda la Identificación de Usuario del SPA9000 del paquete de Registro SIP enviado por el SPA9000. Deberá especificar la dirección IP y el puerto UDP del SPA9000. El SPA400 escucha un requerimiento de Registro SIP del SPA9000
    Dirección Estática Seleccione este botón para ingresar la dirección IP del SPA9000. “Dirección IP” significa que el SPA400 no descubrirá en forma automática la dirección IP y el número de puerto del SPA9000. El usuario debe especificarlo en los campos “Dirección IP” y “Puerto”. También significa que el SPA400 sólo aceptará los mensajes SIP de ésta dirección
    Dirección IP Dirección IP del SPA9000
    Dato - Identificación de Puerto. Cada puerto de línea en el SPA400 puede tener una identificación única para permitir que el SPA9000 identifique las llamadas entrantes desde PSTN
    Identificación de Puerto 1 Identificación del Puerto 1
    Identificación de Puerto 2 Identificación del Puerto 2
    Identificación de Puerto 3 Identificación del Puerto 3
    Identificación de Puerto 4 Identificación del Puerto 4
    Dato - Señalización
    Puerto de Señalización Puerto UDP en el que el SPA400 escucha el requerimiento de configuración de las llamadas entrantes
    Dato - RTP
    Puerto RTP Puerto UDP base que el SPA400 usa para RTP y RTCP. El SPA400 usa un bloque de puertos UDP para enviar y recibir paquetes RTP y RTCP comenzando desde este numero de puerto
    Dato – IP Servicio ToS/Diff
    Paquetes de Señalización de Llamada Campos TOS en el título IP para los paquetes SIP salientes
    Paquetes RTP Campos TOS en el título IP para los paquetes RTP y RTCP salientes
    Dato - Sesión
    Habilitar Timer de Sesión Seleccione este botón si quiere que el SPA400 codifique el titulo del Timer en todas las solicitudes de INVITAR de un tiempo agotado de llamado
    Tiempo de Refrescar Deseado Timer de sesión deseado en segundos
    Tiempo de Refrescar Mínimo Valor mínimo de un timer de sesión, en segundos, que prefiere el SPA400

    Configuración: Voz
    Dato – Codificadores de Voz
    Codificador Preferido Seleccione el codificador de voz preferido que el SPA400 usará para negociar con los dispositivos VoIP remotos para determinar la codificación de voz
    Paquetización Seleccione el tiempo de paquetización para un codificador de voz
    VAD Habilite o deshabilite la Detección de Actividad de Voz en el codificador de voz
    Dato – Timers de Llamados
    Tiempo de Esperar-Respuesta Especifica el tiempo en segundos que el SPA400 debe esperar para que la parte que recibe la llamada responda la llamada. Si ésta parte no responde la llamada dentro de este período de tiempo, la llamada se finaliza automáticamente
    Límite de Llamada Especifica la cantidad máxima de segundos de una llamada. Cuando la duración de la llamada supera éste valor, la llamada es finalizada automáticamente
    Dato – Parámetros de Discado
    Tono termina encendido Especifique el tiempo de encendido del tono en milisegundos para un dígito de discado saliente DTMF
    Tono termina apagado Especifique el tiempo de apagado del tono en milisegundos para un dígito de discado saliente DTMF
    Energía DTMF Nivel de energía en 0.1 dBm de los tonos DTMF que el SPA400 disca hacia los puertos de Telefonía
    Responder luego de Cantidad de llamados que el SPA400 deberá esperar antes de responder una llamada entrante
    Espera de discado hacia afuera Tiempo en milisegundos que el SPA400 debe esperar luego de tomar un puerto de telefonía y antes de discar los dígitos DTMF
    Umbral de batería de discado hacia afuera Antes de tomar un puerto de telefonía para discar, el SPA400 detecta el nivel de voltios en el puerto para asegurarse que el puerto está conectado y disponible. Si el nivel de voltios es inferior a su umbral, el puerto es declarado no disponible
    Dato – Configuraciones de la Línea
    Ganancia de Trasmisión El SPA400 podrá aumentar o atenuar el nivel de energía antes de trasmitir por un puerto de telefonía. Este campo permite que configure el nivel de ganancia en dB
    Ganancia de Recepción El SPA400 podrá aumentar o atenuar el nivel de energía del puerto de telefonía. Este campo permite que configure el nivel de ganancia en dB
    Selección de Impedimento Seleccione el impedimento de la línea que se conecta a los puertos de telefonía del SPA400

    Configuración: Servidor de Correo de Voz
    Dato – Configuraciones del Correo de Voz

    Puerto del Servidor


    Identificación del Usuario SPA9000    		 El puerto UDP por el cual el correo de voz escucha OBS: El número de puerto debe ser diferente del puerto de señalización en la página de interfaz del SPA9000
    La identificación de Usuario del SPA9000 es usada para registrarse en el servidor de correo de voz del SPA400
    Identificación del Suscriptor SPA9000
    Número de depósito de la casilla de correo Número de administración de la casilla de correo    		 La identificación SPA9000 es usada para suscribirse al servidor de correo de voz SPA400 para obtener una notificación
    Número de teléfono usado por el SPA9000 para depositar un correo de voz
    Número de teléfono usado por el SPA900 para acceder a la casilla de correo de voz

    OBS: Las configuraciones en ésta página deben coincidir con la solapa de la Línea # del SPA9000 que se conecta al servidor de correo de voz del SPA400.

    Configuración: Usuarios de Correo de Voz
    Dato – Usuarios de Correo de Voz
    Habilitar Usuario Seleccione este cuadro para habilitar la casilla de correo
    Contraseña de Identificación del Usuario Identificación de casilla de correo
    Contraseña para acceder a la casilla de correo

    OBS: Cada casilla de correo es un Teléfono IP SPA administrado por el SPA9000. Las identificaciones de los usuarios en la página de Correo de Voz de los Usuarios debe ser consistente con la identificación de los usuarios en la casilla de Correo de los Teléfonos IP SPA antes de que el teléfono pueda depositar y acceder al correo de voz.

    Configuraciones extras#

    Con los dispositivos ATA de la marca Linksys hay que configurarle una entrada Dial Plan en particular, ya que si no al cortar desde el telefono conectado a dicho ATA la llamada realmente nunca se corta.

    Dial Plan:

    (<:1206>[2-9]xxxxxx|011xx.|1[2-9]xx[2-9]xxxxxx|xxxx)

    En el modelo SPA 2102 se configura en la opcion de Regional, y en el modelo PAP2T-NA se configura en la opcion para cada usuario SIP (Menu User 1 y User 2)

    PAP2T-NA

    y poner enable ip dialing en yes
